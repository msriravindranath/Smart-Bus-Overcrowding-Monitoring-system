System Architecture

  • Dual Ultrasonic Sensors: Placed at the entrance and exit for precise bi-directional counting.
  • ESP32 Microcontroller: The core processing unit handling raw sensor data efficiently.
  • Direction-Based Detection: Smart algorithm distinguishes between passenger entry and exit securely.
  • Real-Time Counting: Instant passenger tally updates directly visible on the monitor.
  • WiFi Web Monitoring: Live, seamless data transmission from the bus to this dashboard layout.

Future Scope

  • AI-Based Multi-Person Detection: Vision-based confirmation using advanced ML camera systems.
  • Multi-Sensor Grid: Extended sensor coverage specifically optimized for wide-door bus configurations.
  • Cloud Integration: Centralized RTC monitoring across the complete public transportation fleet.
