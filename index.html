System Architecture

  • Dual Ultrasonic Sensors: Placed at entrance & exit for bi-directional counting.
  • ESP32 Microcontroller: Handles real-time sensor processing.
  • Direction-Based Detection: Identifies entry and exit accurately.
  • Real-Time Counting: Instant passenger updates.
  • WiFi Web Monitoring: Live data transmission to dashboard.

Future Scope

  • AI-Based Detection: Camera-based multi-person validation.
  • Multi-Sensor Grid: Optimized for wide-door buses.
  • Cloud Fleet Monitoring: Centralized RTC tracking system.
