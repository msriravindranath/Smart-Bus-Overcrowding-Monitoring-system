System Architecture

  • Dual Ultrasonic Passenger Detection: Two ultrasonic sensors are installed at the entrance and exit of the bus door. These sensors monitor passenger movement through distance detection.
  • Directional Passenger Flow Detection: Sensor activation order determines whether a passenger enters or exits the bus.
  • ESP32 Edge Controller: The ESP32 processes sensor signals, manages passenger counts, and sends data to the cloud.
  • Cloud Storage using Google Sheets: Passenger occupancy data is stored in Google Sheets through Apps Script APIs.
  • Real-Time Dashboard: Users can monitor bus occupancy and availability from a web dashboard.
  • Smart Route Planner: Passengers can search routes between cities and view available buses and fares.

Future Scope

  • AI Passenger Detection Computer vision models could improve passenger counting accuracy.
  • GPS Bus Tracking Real-time GPS tracking can show the exact location of buses.
  • Passenger Mobile App Passengers could check seat availability and routes from mobile apps.
  • Demand Prediction Machine learning models can predict peak travel times.
